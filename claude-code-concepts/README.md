# Claude Code Concepts Course

A guided course on Claude Code fundamentals -- 12 concepts that teach you how Claude Code actually works, from architecture to advanced automation. Whether you're a developer or a non-technical power user, start from Tier 1 and work down. Each concept builds on the previous ones.

**Teaching others:** Share this README as the starting point. Each note is self-contained but cross-referenced.

**Quick reference:** Jump to any concept directly. Each note has a Table of Contents for navigation.
