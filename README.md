# Terrium

A knowledge terrarium — curated insights, concepts, and resources on AI-assisted workflows.

By [Terry Chan](https://github.com/terry42hk)

## What's inside

### [Claude Code Concepts](claude-code-concepts/)

A structured 12-concept course on mastering Claude Code, from foundational mental models to advanced orchestration patterns. Organized in three tiers:

- **Foundation** — Harness vs Model, Agent Loop, Tools, Permission Model
- **Configuration** — Context Window, CLAUDE.md, Verification, Subagents
- **Scaling** — Context Management, Hooks, Skills, MCP

Also includes a [glossary](claude-code-concepts/glossary.md) and [extended discussions](claude-code-concepts/extended/) that go deeper on specific topics.

## Who this is for

Anyone working with AI coding agents — whether you're just getting started with Claude Code or looking to level up your mental models. The content focuses on *concepts over commands*: understanding the "why" behind effective AI-assisted workflows.
